Detailed Description

A key value pair used by the KeyValueFileParser class it stores the key in four formats: as wide unicode and "normal" 8 bit strings both in uppercase letters and in the way the were discovered in the file the values are stored in the same four string formats as well as as int, unsigned int, float, and vectors in case of list values
