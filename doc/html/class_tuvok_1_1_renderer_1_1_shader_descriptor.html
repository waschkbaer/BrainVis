Detailed Description

A common way to identify a shader. We need to know whether a shader is equivalent. When we're asked to load a new shader, we search through a list of cached ones and see if it already exists. This allows us to avoid loading it multiple times, and more importantly compiling it multiple times. This is made a bit difficult by how we build shaders: they can come from files or strings.
